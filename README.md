# Work Calc Pro

UK Wage Calculator for Android.

## Features
- Multiple employer support
- Hourly rate + overtime rate configuration
- Calendar view with shift management
- UK Income Tax calculation (2024/25)
- National Insurance calculation
- Universal Credit deductions (55% taper)
- PDF payslip export
- Dark theme, Material Design 3

## Built With
- Kotlin + AndroidX + Room
- Material Design 3
- Coroutines
- iText PDF
